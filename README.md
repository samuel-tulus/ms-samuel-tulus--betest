# ms-samuel-tulus- betest

## Requirements :
```
1. Node JS
2. MongoDB
3. Redis Server
```

## Preparation :
```
1. Move config folder to the outside of project folder in the same directory
2. Rename and remove ".example" for .env.example
3. Fill the config according to your machine
```

## Setup :
```
1. npm install
```

### Run :
```
1. npm start
```
